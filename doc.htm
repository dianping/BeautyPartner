接口001 名称解释：
储值卡余额查询（调出） 调出： 点评为客户端，调用第三方为服务端
接口描述 调入： 点评为服务端，第三方push消息进来
API名称： 会员卡余额查询 card/query_balance
API描述: 根据cardId查询储值卡余额
数据格式: JSON
方法: HTTP GET
请求URL http://xx.xx.xx/card/query_balance
请求参数
参数 必须 类型 描述 示例
channle 是 int 合作方标识 1
sign 是 String 签名
card_id 是 String 会员卡ID xxx
返回结果
参数 必须 类型 描述 示例
status 是 String 状态：SUCC, 或者 FAIL
error_code 是 String 如果status返回FAIL，这里返回对应的错误码
error_reason 是 String errorCode对应的错误说明
type 是 String 消费类型。可选值：CONSUME_AMOUNT(金额消费)；CONSUME_TIME(次数消费)
amount 是 String type=1 金额，保留两位小数 ;type=2,返回次数
内容加密		 100.12
请求示例
http://xx.xx.xx/card/query_balance?channel=1&sign=xxx&card_id=xxx
返回示例：
{
    "status": "ok",
    "error_code":"",
    "error_reason":"",
    "type":"CONSUME_AMOUNT",
    "amount": "100.12"
}

接口003

error_code 否 String 如果status返回FAIL，这里返回对应的错误码
error_reason 否 String errorCode对应的错误说明
data 是 List 返回数据，具体如下 　
type 是 Int 1-条形码；2-二维码

接口005
会员基本信息获取（调出）
接口描述
API名称： 会员基本信息获取 card/query_members
API描述: 批量获取会员信息，根据时间戳（增量标识）增量获取。
数据格式: JSON
方法: HTTP GET
请求URL http://xxx.xxx.xxx/card/query_members
请求参数
参数 必须 类型 描述 示例
channle 是 int 合作方标识
sign 是 String 签名（参数按字典序排序，MD5对参数加密）
update_time 是 Date 时间戳，返回当前时间戳之前更新的数据,格式:
yyyy-MM-dd HH:mm:ss
page_no 是 int 页码。取值范围:大于零的整数; 默认值:1
page_size 是 int 每页条数。取值范围:大于零的整数; 默认值:40;
最大值:100
返回结果
参数 必须 类型 描述 示例
status 是 String 状态：SUCC, 或者 FAIL
error_code 否 String 如果status返回FAIL，这里返回对应的错误码
error_reason 否 String errorCode对应的错误说明
total_results 是 Int 返回总记录的行数 100
has_next 是 Boolean 是否存在下一页 TRUE
data 是 List 会员数据，具体如下 　
user_id 是 int 会员ID
card_id 是 String 储值卡Id
mobile 是 String 会员手机号码
name 是 String 会员姓名
address 是 String 会员地址
create_time 是 String 开卡时间，格式：yyyy-MM-dd HH:mm:ss
birthday 是 String 持卡人生日
member_card 是 List 储值卡信息，具体如下 　
card_type 是 String 储值卡类型 金卡／银卡
consume_type 是 String 消费类型，1-金额，2-次数
card_scope 是 String 适用范围。Eg:全店通用，仅剪发
card_status 是 String 卡状态。可选值：ENABLE(可用)，DISABLE(不可用)
related_shop 是 List 储值卡所属门店，具体如下 　
shop_id 是 String 门店ID
shop_name 是 String 门店名称
请求示例
http://xx.xx.xx/card/query_members?channel=1&sign=xxx&update_time=
返回示例：
{
    "status": ok,
    "amount": "100.12"
}
